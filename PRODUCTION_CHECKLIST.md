### 🚀 طرق النشر المتاحة

#### 1. النشر التقليدي
```bash
npm install
npm start
```

#### 2. النشر بـ Docker
```bash
docker-compose up -d
```

#### 3. النشر بـ PM2
```bash
npm install -g pm2
pm2 start server.js --name "tamsik"
```

#### 4. النشر التلقائي
```bash
npm run deploy
```

### 🔧 إعدادات الإنتاج المطلوبة

#### ملف .env
```env
NODE_ENV=production
PORT=3000
DB_TYPE=sqlite
JWT_SECRET=your_secure_jwt_secret_here
CORS_ORIGIN=*
```

#### إعدادات Nginx (اختياري)
```nginx
server {
    listen 80;
    server_name your-domain.com;
    location / {
        proxy_pass http://localhost:3000;
    }
}
```
